FRIENDLY INTUITIVE UI

Every application of mine aims at improving two main aspects of a business:

  • its productivity and efficiency in a given area as a whole.

  • the productivity and efficiency of the team using the software.

Thus, a friendly user interface is a pivotal aspect of each of my projects. To enhance the user experience, the look and feel of a software aims at being familiar – if not better – to what the user is already using in the business. The user-interface will also reflect your brad’s identity and any further customisation will be up to the client to decide.

In addition, the UI’s design will include some simplifications to automate as much tasks as possible. These can be simple things going from ensuring that the first letter of a name is capitalise to more complex scenario like sending multiple emails at the same time. This is to reduce chances of human’s errors and let the user focussing on doing their daily tasks without stressing about potential “irreversible” mistakes.

Despite each business being different – and to some extent unique – the application’s design and its promises are accomplished by understanding how the user usually work. This means I take care of understanding what the daily operations are together with their challenges and common annoyances. Together we can make the your work-flow easier and headache-free.

OFFICE PACK INTEGRATED

Microsoft Office is an essential in every office. Despite having your own software you will still need to rely on these family of Microsoft’ products. Thankfully, Microsoft offers way to interact with each of their product which means that your tailored software can interact with them too. Based on the business needs, we can automate whatever process you might be doing with these products.

Here some of the most common features that I've integrated:

Read and write Excel's Spreadsheets:

  • Excel files can be read by the application and if necessary their content can be stored inside the application itself.

  • If you have information that required to be organised into an Excel file, this can be done including formatting and formulas. This is also known as Reporting's functionalities that allows you to organise your data and get them ready to be shared with other departments within your organisation.

Write Word documents:

If you have data that must be organised in a template Word document, they can be exported with just one click. You will not need a template file stored somewhere in your network; it will be written by the Application. Stop worrying about copying and pasting chunks of information. Reduce the stress and the anxiety of making mistakes in the files you send to your customers or managers.

Send emails with Outlook:

This function will include a section where you can save your contacts and send emails with a click. Emails can have either a predefined text or one of your own for each email. Attachments can also be predefined or decided upon the situation. Moreover, you can send multiple emails with one click. You can also create spreadsheets and Word documents on the go and send them by email.

N.B. Office must be installed in your computer.

PDF REPORTS

PDF’s are another essential in every office. Nowadays, Microsoft have put to its users an easy way to convert documents to PDF files. This is done through a PDF printer which is by default installed on every Window’s computer. In the unlikely event this is not installed, it can be easily downloaded and installed.

By using the same technology, your Application can produce PDF files which can be saved on your computer and/or sent by email. The Application will have a Report's Viewer window which allow the user to view the PDF document. The document can then be either saved on the computer or sent by email via outlook.

MULTI-USER

Depending on the need of your business, the Application can be designed for either one or more users. Every software will rely on a database to store its data which will be either on your local computer or on a network.

For single-user applications, might be used a SQLite database which will be installed in a safe location of your computer. The application will then exchange requests with the database in order to fetch and amend flows of information. Alternatevely, a Microsoft Access database can be used to store the data. This approach does not require an internet connection in order to work.

However, an online database such as MySQL is the most secure option to protect your data from hacking attempts on your computer.

As to multi-user applications, they will need an online database such as MySQL. Applications running with a MySQL database as their back-end will require to rent a server provider in order to host the database online.
